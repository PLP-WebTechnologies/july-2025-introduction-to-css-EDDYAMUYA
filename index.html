Introduction to CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

The CSS Box Model

All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element.

Common CSS Properties

Text Properties

CSS provides many properties for styling text, including:

  • color: Sets text color
  • font-family: Defines the font
  • font-size: Controls text size
  • text-align: Aligns text

Box Model Properties

Properties that control the box model:

  • margin: Space outside the border
  • padding: Space inside the border
  • border: Edge around padding and content
  • width/height: Content dimensions

Background Properties

Properties for styling element backgrounds:

  • background-color: Sets solid color
  • background-image: Adds images
  • background-position: Positions images
  • background-repeat: Controls tiling
